What is jQuary?

Jquary is a fast, small, and feature-rich JavaScript library. It makes things like HTML document travelsal and manipulation, event handling, animation, and Ajax much simpler with and easy-to-use API that works across a multitude of vrowsers. With a combination of versatility and extensibility, jQuary has changed the way that millions of people write JavaScript.
